Exercise 1: File Organization Automation Script

Objective: Help junior interns practice process automation by creating a script that organizes files based on their types.

Instructions:

  1. Task Overview:
  2. You are responsible for writing a simple automation script that organizes files into folders based on their type. This will help you practice basic scripting and automation.
  3. Scenario:

    4. You receive daily batches of files (images, PDFs, documents) that need to be sorted into folders. Your task is to create a script that:

    • Scans a folder for files.
    • Automatically creates folders named "Images," "Documents," and "Miscellaneous."
    • Moves files into the appropriate folder based on their type.
  4. Steps:

    5. Write a Python (or Bash) script that:

    • Scans a directory for files.
    • Creates the necessary folders if they don’t exist.
    • Sorts the files into their respective folders based on the file type.

Code Example:


import os
import shutil

# Define source folder
source_folder = 'path_to_your_folder'

# Create destination folders if they don't exist
if not os.path.exists('Images'):
    os.mkdir('Images')
if not os.path.exists('Documents'):
    os.mkdir('Documents')
if not os.path.exists('Miscellaneous'):
    os.mkdir('Miscellaneous')

# Iterate over files in the folder
for filename in os.listdir(source_folder):
    if filename.endswith('.jpg') or filename.endswith('.png'):
        shutil.move(os.path.join(source_folder, filename), 'Images')
    elif filename.endswith('.pdf') or filename.endswith('.docx'):
        shutil.move(os.path.join(source_folder, filename), 'Documents')
    else:
        shutil.move(os.path.join(source_folder, filename), 'Miscellaneous')

Checklist:

  • The script should scan the folder and move files correctly.
  • The "Images," "Documents," and "Miscellaneous" folders should be automatically created if they don’t exist.
  • Be prepared to explain how your script works in a short write-up.

Submission:

Submit your script and a 2-3 sentence explanation of what it does and how you wrote it.
